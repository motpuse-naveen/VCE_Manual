Special hydraulics

Version 1A

Hydraulic function, 4th switchable from 3rd

(optional equipment)

This equipment serves one hydraulic function at the time, 3rd or 4th. It is used when two auxiliary hydraulic functions are needed, for example road sweeper with hydraulic slewing device.

  • Changing from 3rd to 4th hydraulic function = Pressing and holding the push button (arrow).
  • Changing back from 4th to 3rd hydraulic = Releasing the push button (arrow).
Version 1B

Single control lever version 1B

Control lever version 1B provides flow adjustment, see page 66. This is affecting also the 4th hydraulic function, when changing from 3rd to 4th.

This version provides common flow adjustment for hydraulic function 3rd or 4th, depending on what is selected. For further information hydraulic function 3rd or 4th.
